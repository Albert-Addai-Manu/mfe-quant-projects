# mfe-quant-project
Quantitative finance projects for Master of Financial Engineering applications--Black-Scholes pricer.
# motivation
The Black-Scholes model is foundational, but its constant volatility assumption fails to capture market realitites 
## Overview
This repository provides a Python implementation of the classic Black-Scholes formula for pricing European options. Key features:
-Accurate calculation of call and put prices
-Input validation and clear error handling
-Readable and fully commented code
